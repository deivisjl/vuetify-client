## Environment setup

### First steps
Create a local environment file (`cp .env.example .env`). Set your credentials of the rest-api
### Install dependencies
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```
